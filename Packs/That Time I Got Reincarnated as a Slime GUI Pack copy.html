That Time I Got Reincarnated as a Slime GUI Pack

Introduction

This is a GUI-only resource pack themed after the show "That Time I Got Reincarnated as a Slime." You can freely use it with other resource packs by placing it on top of your existing pack. The pack features a customized palette of blue, green, and yellow, with custom images for each container, including the hot bar. Future updates may include different color schemes, such as red and purple. If you find any bugs, please report them so I can fix them promptly.
