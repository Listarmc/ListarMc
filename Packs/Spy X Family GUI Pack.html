Spy X Family Custom GUI Pack

Introduction

This is a GUI-only resource pack themed towards the show "Spy X Family". You can freely use it with other resource packs by placing it on top of the resource pack you use. I have changed the hot bar and experimented with different pictures in each container. The color palette includes Blue and Pink, but I'm open to suggestions for other colors like Gray and White. If you have any preferences, let me know!

If you find any bugs, please report them to me, and I'll fix them as soon as possible.
